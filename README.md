->  **Financial Dashboard built in Power BI**!

This dashboard brings together **Crypto, Stocks, and ETFs** into one interactive platform, making market analysis easier and visually insightful.

 **Key Features of the Power BI Dashboard:**
• Interactive candlestick charts
• Category switching: Crypto / Stocks / ETFs
• Time-range filters (1Y, 3M, 5D, 6M, YTD, Max)
• Dynamic KPIs with up/down indicators
• Light & Dark mode theme integration
• Scrollable asset list with icons
• Clean, modern UI for fast decision-making

 **What I used in Power BI:**
• Advanced DAX measures for calculations
• Bookmarks for theme switching
• Custom visuals (candlestick chart)
• Conditional formatting for KPIs
• Dynamic buttons and navigation
• Image integration for logos/icons
• Slicers for time-range selection

 **What I learned:**
• Creating a full financial dashboard in Power BI
• Building user-friendly navigation systems
• Using DAX for real-time calculations
• Applying custom themes for UI/UX improvement
• Integrating live financial data visuals effectively

Power BI is extremely powerful for building professional dashboards, and this project helped me explore deeper visualization techniques.
